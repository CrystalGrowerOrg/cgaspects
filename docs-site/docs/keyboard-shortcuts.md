# Keyboard Shortcuts

All shortcuts can be viewed and customised in **Help → Keyboard Shortcuts** (`Ctrl+/`).

---

## File

| Action | Shortcut |
|--------|----------|
| Import | `Ctrl+I` |
| Import CSV from Clipboard | `Ctrl+Shift+C` |
| Render viewport | `Ctrl+R` |
| Export XYZ | `Ctrl+Shift+E` |

## Customizing Shortcuts

1. Open **Help → Keyboard Shortcuts** (`Ctrl+/`)
2. Find the action in the list (use the search box to filter)
3. Click on the current shortcut to edit it
4. Press the new key combination
5. Click **Save** or **Apply**

Changes are saved to the application configuration and persist across sessions.

Click **Reset to Defaults** to restore all shortcuts to their original values.
